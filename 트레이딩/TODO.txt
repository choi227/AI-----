----------------------------------------------------------------
 [1주차] 기반 구축
----------------------------------------------------------------

[ ] 1. Gemini API 키 발급
    - https://aistudio.google.com 접속 → "Get API key" 클릭
    - 발급받은 키를 .env 파일에 저장

[ ] 2. 개발 환경 세팅
    - Python 가상환경 생성 (venv)
    - 필수 패키지 설치
      · google-generativeai   (Gemini LLM)
      · chromadb              (Vector DB)
      · langchain             (AI 파이프라인)
      · fastapi + uvicorn     (백엔드)
      · streamlit             (대시보드)
      · yfinance              (반도체 주가 데이터 - 무료)
      · pandas + requests     (데이터 처리)
      · python-dotenv         (API 키 관리)
    - .env 파일 생성 (API 키 저장용)
    - .gitignore 파일 생성 (.env 제외)

[ ] 3. 프로젝트 폴더 구조 생성
    trading_system/
    ├── .env                  ← API 키
    ├── .gitignore
    ├── requirements.txt
    ├── data/                 ← 수집한 원본 데이터 저장
    ├── database/             ← ChromaDB 저장 위치
    ├── src/
    │   ├── collector.py      ← 반도체 주가 데이터 수집 (yfinance)
    │   ├── processor.py      ← 기술 지표 계산
    │   ├── rag.py            ← RAG 시스템
    │   ├── agent.py          ← AI 에이전트
    │   └── api.py            ← FastAPI 백엔드
    └── dashboard/
        └── app.py            ← Streamlit 대시보드

[ ] 4. 데이터 수집 모듈 구현 (collector.py)
    - yfinance로 반도체 종목 주가 수집
      · NVDA, AMD, INTC, ASML, TSM, 000660.KS
      · SOXX, SMH ETF
    - 수집 데이터를 CSV로 저장
    - 일별/주별 자동 수집 구조 설계

[ ] 5. 데이터 전처리 모듈 구현 (processor.py)
    - 수집 데이터 정제 및 결측치 처리
    - 기술 지표 계산
      · 이동평균선 (MA5, MA20, MA60)
      · RSI (과매수/과매도 판단)
      · 볼린저 밴드
      · 거래량 분석
    - 정형 데이터 → 텍스트 요약 변환 (LLM 입력용)

----------------------------------------------------------------
 [2주차] AI 핵심 기능
----------------------------------------------------------------

[ ] 6. RAG 시스템 구현 (rag.py)
    - ChromaDB 연결 및 컬렉션 생성
    - 반도체 시장 분석 텍스트 → 임베딩 → Vector DB 저장
      (업황 요약, 지표 해석, 과거 패턴 등)
    - 쿼리 입력 시 관련 문서 검색 기능

[ ] 7. AI 에이전트 구현 (agent.py)
    - Gemini API 연결
    - 시장 진입 신호 판단 Agent
      · 현재 주가 + 기술 지표 + RAG 검색 결과 종합
      · 매수 / 매도 / 관망 신호 + 근거 출력
    - 리스크 감지 Agent
      · 섹터 전반 하락 감지
      · 개별 종목 이상 패턴 감지
      · 리스크 수준 (낮음 / 중간 / 높음) 판단
    - 트레이딩 전략 추천 Agent
      · 위 두 Agent 결과를 종합
      · 종목별 투자 전략 및 근거 텍스트 생성

[ ] 8. FastAPI 백엔드 구현 (api.py)
    - GET /market-data     : 반도체 종목 최신 시세 반환
    - GET /signal          : AI 매수/매도 신호 반환
    - GET /risk            : 리스크 감지 결과 반환
    - POST /chat           : 챗봇 질의응답 엔드포인트
    - API 동작 테스트 (Swagger UI 확인)

----------------------------------------------------------------
 [3주차] 대시보드 + 마무리
----------------------------------------------------------------

[ ] 9. Streamlit 대시보드 구현 (app.py)

    - AI 매수/매도/관망 신호 표시
    - 리스크 수준 게이지
    - 종목별 AI 트레이딩 전략 추천 텍스트
    - 챗봇 인터페이스 ("NVDA 지금 사도 될까?" 같은 질문)

[ ] 10. 통합 테스트
    - 전체 파이프라인 end-to-end 테스트
    - 에러 처리 (API 오류, 데이터 없음 등)

[ ] 11. 포트폴리오 문서 작성
    - README.md 작성
      · 시스템 개요 및 아키텍처 다이어그램
      · 설치 및 실행 방법

    - 시연 영상 또는 스크린샷 캡처
